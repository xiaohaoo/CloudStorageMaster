## CloudStorageMaster

云存储工具服务(Cloud Storage Service)，目前仅支持**阿里云OSS**。

### 功能

- [x] 上传文件
- [x] 下载文件
- [x] 删除文件
- [x] 文件断存在

### 使用

```groovy
implementation group: 'com.xiaohaoo', name: 'oss-provider', version: '1.1.0'
```

```xml

<dependency>
    <groupId>com.xiaohaoo</groupId>
    <artifactId>oss-provider</artifactId>
    <version>1.1.0</version>
</dependency>

```
