## oss-provider

云对象存储OSS（Object Storage Service）工具类，仅支持**阿里云**。

### 功能

- [x] 上传文件
- [ ] 下载文件
- [ ] 文件管理

### 使用

```groovy
implementation group: 'com.xiaohaoo', name: 'oss-provider', version: '1.1'
```

```xml

<dependency>
    <groupId>com.xiaohaoo</groupId>
    <artifactId>oss-provider</artifactId>
    <version>1.1</version>
</dependency>

```

#### 构建

```
./gradlew build
```
